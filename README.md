# ICT4D2021-RadioMaliVoterBackend
This is the system's backend by ICT4D 2021's group three.\
This repository intents to provide a clean version, ready to be forked, set up and run using Heroku.
##Installation
Fork this repository.\
Fill in the database fields on lines 12-15 and 55-58.\
Update your forked repository.\
Create a Heroku app.\
Link your forked repository.

The application will now build and start running in your app container.

##Usage
Change the address of your VoiceXML files to the external address of your Heroku endpoint and the backend will handle everything automatically.
